# fastapi sandbox

## Preparing and running the app

* `make venv`
* `make install`
* `make seed`
* `make server-dev`

## Linting and testing

* `make lint`
* `make test`
* `make test-watch`

## Other useful commands

* `pip list --outdated`
* `pip-compile --upgrade-package pytest`
