# fastapi sandbox

## Preparing and running the app

* `make venv`
* `make install`
* `make seed`
* `make server-dev`
* `curl http://127.0.0.1:8000/projects`

## Linting and testing

* `make lint`
* `make test`
* `make test-watch`

## Other useful commands

* `pip list --outdated`
* `pip-compile --upgrade-package pytest`
